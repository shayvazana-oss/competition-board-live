# לוח תוצאות התחרות — מצב לייב

עותק עצמאי של פלטפורמת הניקוד, מתארח ב-GitHub Pages תחת החשבון `shayvazana-oss`.

מתחבר לאותו מסד **Firebase Realtime Database** של התחרות — כך שהזנת נקודות כאן משתקפת בזמן אמת בכל המסכים המחוברים (כולל הגרסה המקורית).

## מצב לייב 🔴

כפתור **"מצב לייב"** (פינה שמאלית-תחתונה, אחרי הכניסה) פותח לוח תוצאות במסך מלא:

- הקבוצות מסודרות לפי הניקוד, השורות **מחליקות ומחליפות מקום** בכל שינוי (אפקט יורוויזיון), עם מדליות 🥇🥈🥉.
- **סרגל הזנת נקודות** בתחתית: בחירת קבוצה + כפתורים מהירים, סכום חופשי, הורדה, וביטול (undo). כל הזנה מופיעה מיד אצל כל הצופים.
- "מסך מלא" למקרן, "הסתר בקרה" (או `C`) לתצוגה נקייה, `Esc` לסגירה.

## מבנה

- `index.html` — האפליקציה המלאה (React ארוז) כולל מודול הלייב.
- `_live_board.js` — קוד המקור של מודול הלייב.
- `_inject.js` — הזרקה מחדש של המודול ל-`index.html` (`node _inject.js`).

הפריסה ל-GitHub Pages אוטומטית בכל push ל-`main`.
